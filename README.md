Калькулятор — задание 24.2.4
ростой калькулятор с четырьмя арифметическими операциями. Включает позитивные тесты для каждой операции

Структура проекта
`calculator.py` — реализация класса Calculator
`tests/test_calculator.py` — тесты для калькулятора
`README.md` — документация

Как запустить тесты
Убедитесь, что установлен Python
Откройте терминал и перейдите в папку проекта
Запустите тесты
python -m unittest tests/test_calculator.py
